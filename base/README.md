# MagiskFakehttp
> [FakeHTTP](https://github.com/MikeWang000000/FakeHTTP) is a tool used to Obfuscate all your TCP connections into HTTP protocol.

> [MagiskFakehttp](https://github.com/Droid-MAX/magisk-fakehttp) lets you run fakehttp on boot with [Magisk](https://github.com/topjohnwu/Magisk)

## Supported architectures
- `arm32v7`, `arm64`, `i686`, `x86_64`

## How fast are fakehttp updates?
Instant! This module is hooked to the official FakeHTTP build process

## For issues and more information, check out the [main project repo](https://github.com/Droid-MAX/magisk-fakehttp)
